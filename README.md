# FUTURE_FS_02 – Client Lead Management System (Mini CRM)

This project is a **Client Lead Management System (Mini CRM)** developed as part of  
**Future Interns – Full Stack Development Task 02**.

The system allows an admin to securely manage client leads received from a website,
track their status, and update them through a dashboard.

---

## Features

### Admin Authentication
- Secure admin login using JWT
- Protected routes
- Logout functionality

###Lead Management
- View all client leads
- Track lead details (Name, Email, Source, Status)
- Update lead status (New → Contacted → Converted)

### Frontend Dashboard
- Built with React.js
- Clean and user-friendly UI
- Color-coded lead status
- Real-time updates

###Backend API
- Node.js & Express.js
- REST APIs
- MySQL database
- JWT-based security

##Setup Instructions

### Backend
cd crm-backend
npm install
npm run dev


### Frontend
cd crm-frontend
npm install
npm start


Frontend runs on `http://localhost:3000`  
Backend runs on `http://localhost:5000`
